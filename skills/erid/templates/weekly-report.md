### Template Usage Notes
This template is designed to be generated automatically from ERID data. Key sections pull from:
- `v_active_pipeline` view for pipeline dashboard
- `v_weekly_actions` for action items
- `v_recent_ciso_signals` for market intelligence
- `v_firm_relationships` for search firm status

### Customization
Add company-specific metrics, personal KPIs, or additional sections as needed. The template variables ({{VARIABLE_NAME}}) should be replaced with actual data from your ERID database.

### Report Frequency
- **Weekly:** Every Friday afternoon
- **Monthly:** Last Friday of the month (comprehensive)
- **Quarterly:** Strategic review with goals adjustment

### Distribution
- **Primary:** Personal review and planning
- **Secondary:** Mentor/coach (if applicable)
- **Archive:** ERID reports directory for trend analysis
